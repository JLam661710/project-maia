# 🔭 North Star: Maia

## Core Value Proposition
打造一个**“主动响应式”**的 AI 访谈系统，通过严密的逻辑编排（Analyst -> Judge -> Interviewer），在对话中深度挖掘用户需求，最终交付高价值的结构化解决方案。

## Key Principles
1.  **Logic First**: 逻辑严谨性优于响应速度（Accept Latency for Quality）。
2.  **Strict Orchestration**: 采用全同步阻塞流，确保每一步都有据可依。
3.  **Transparency**: 向用户透明展示 Agent 的思考过程，建立信任。
4.  **Structure**: 一切非结构化对话最终都必须坍缩为结构化的 `InterviewState`。
